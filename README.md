# 🚀 Kaushal Singh - Portfolio

A modern, responsive personal portfolio website built using React, TypeScript, and Vite.
It showcases my projects, skills, and provides a working contact form for direct communication.

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/kaushals1950/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
